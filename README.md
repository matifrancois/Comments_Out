# Comments Out

This repo has the code of the visual studio code extension Comments Out, this is an extension that allows you to clear all the comments in your code.

## Languajes support

- C
- C++
- C#
- JavaScript
- Java
- Perl
- PHP
- Python
- R
- Ruby
